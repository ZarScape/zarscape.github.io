# ZarScape - Developer Portfolio

This is a professional developer portfolio designed to showcase projects, skills, and experience in a clean, modern, and visually appealing way. It's built with Next.js, TypeScript, and Tailwind CSS, and it dynamically fetches profile information and projects from the GitHub API to keep the content up-to-date.

![Portfolio Screenshot](https://raw.githubusercontent.com/ZarScape/ZarScape/refs/heads/main/images/other/portfolio-preview.png)

## Features

- **Dynamic Content**: Automatically fetches and displays the latest profile information (name, bio, location, avatar) and public repositories from the GitHub API.
- **Responsive Design**: Fully responsive layout that looks great on all devices, from mobile phones to desktops.
- **Modern UI/UX**: Sleek and modern design built with **ShadCN UI** and styled with **Tailwind CSS**.
- **Interactive & Animated**: Features smooth page transitions, a custom cursor, animated background gradients, and a scrolling skills section for an engaging user experience.
- **Project Showcase**: Displays projects in a clean grid layout with links to GitHub, live demos, and downloads.
- **"Load More" Functionality**: Dynamically loads additional projects from GitHub without a page refresh.
- **Sticky Header**: Provides persistent navigation as the user scrolls through the page.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [TailwindCSS Animate](https://tailwindcss.com/docs/animation) & [Framer Motion](https://www.framer.com/motion/) (implicitly via ShadCN)
